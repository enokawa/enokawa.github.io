【備忘録】githubへのコミット

Jan 6, 2014   #git  #github 

gitのローカルリポジトリからgithubへのコミット方法

環境 MacBook Pro 13inch (Mid 2012) OS X 10.8.5 iTerm2

まずローカルディレクトリに移動

cd git/hoge/

git 開始

git init

フォルダ以下全部追加

git add *

ローカルにコミット

git commit -m 'first commit'

リモートレポジトリ originを追加

git remote add origin https://github.com/user/hoge.git

githubにpush

git push origin master
